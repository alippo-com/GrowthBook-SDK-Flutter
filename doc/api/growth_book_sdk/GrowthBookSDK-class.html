GrowthBookSDK

GrowthBookSDK class Null safety

The main export of the libraries is a simple GrowthBook wrapper class that takes a Context object in the constructor. It exposes two main methods: feature and run. It also includes stream of StateHelper which can be utilized in case sdk is not loaded yet and we want to show some in place.
